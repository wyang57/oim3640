<!-- My Project Proposal -->

<!-- What I'm building:   -->
A Flask application that accepts a place name and returns the nearest MBTA station.

<!-- Why I chose this:   -->
The project is required for the course, and it guides me through a sequence that links user input to outside information. I want to understand how that flow works in a web context.

<!-- Core features:   -->
- A form that accepts a place name  
- A function that sends the place name to the Mapbox API  
- A function that uses the coordinates to query the MBTA API  
- A template that shows the station name and its accessibility  
- A map that marks the location and the station

<!-- What I don't know yet:   -->
- How to structure Flask routes for GET and POST  
- How to embed a Mapbox map inside a template  
- How to read MBTA API responses  
- How to manage API keys with environment variables
