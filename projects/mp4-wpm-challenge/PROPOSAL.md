<!-- MP4 Proposal — WPM Challenge Game -->

<!-- What I'm Building -->
A terminal-based typing speed game that measures how fast and accurately a user can type a given sentence. The program will calculate WPM, accuracy, and total time, then compare the user's performance to my benchmark of 90 WPM at 100% accuracy.

<!-- Why I Chose This -->
I consider myself a fast typer, and I’ve spent a lot of time on typing‑race sites like Typemonkey to measure my speed. I’m semi‑consistently around 90 WPM with 100% accuracy, so building a typing challenge that mirrors the tools I’ve used feels both fun and relevant to my interests.

<!-- Core Features -->
- Display a random sentence for the user to type
- Start and stop a timer based on user input
- Calculate:
  - Words per minute (WPM)
  - Accuracy percentage
  - Total time
- Compare results to my benchmark (90 WPM, 100% accuracy)
- Provide humorous feedback based on performance

<!-- Stretch Features -->
- Multiple difficulty levels (short, medium, long sentences)
- A leaderboard saved to a file
- A “practice mode” with unlimited attempts
- Color-coded output for accuracy and speed

<!-- What I Don’t Know Yet -->
- Best way to measure accuracy character-by-character
- How to structure timing cleanly across functions
- Whether to store results in a file or keep it simple
