<!-- MP5 Proposal Job Quiz 2.0 -->

<!-- What I'm Building -->
I am creating a second version of my job quiz. This version will use a more flexible scoring approach and pull career information from the O*NET Web Services API. The quiz will shape a profile from the user’s answers and use that profile to request matching occupations from the API.

<!-- Why I Chose This -->
My first job quiz helped me understand how to structure a small project. It also gave me a tool that I would actually use. I think other students would use it too. The idea was simple once I figured out the flow. I want to build a stronger version now that I understand more. I want something that feels more helpful and more advanced.

<!-- Core Features -->
The quiz will use weighted scoring.  
The quiz will create a profile from the user’s answers.  
The program will request matching occupations from the O*NET API.  
The program will show a summary that explains the match.

<!-- Stretch Features -->
I may add a way to save results.  
I may add a small chart that shows the user’s profile.

<!-- What I Don’t Know Yet -->
