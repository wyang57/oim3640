<!-- MP5 Job Quiz 2.0 -->

This project expands the idea from my first job quiz. The new version will use weighted scoring and career data from the O*NET Web Services API. The goal is to create a quiz that feels more helpful and more flexible.

<!-- Features -->
The quiz will shape a profile from the user’s answers.  
The program will request occupations from the API.  
The program will explain why each match fits the user.
