<!-- Mini Project 1 — Job Suitability Quiz -->

This project is a simple Python quiz that recommends a job category based on a user’s preferences. The program asks a series of yes/no questions, assigns points to different job categories, and outputs the best match.

<!--  How to Run -->
1. Open a terminal in this folder.
2. Run the program: python app.py

3. Answer the questions by typing "yes" or "no".

<!--  Features -->
- Asks the user several preference‑based questions
- Scores answers across multiple job categories
- Recommends a job type based on the highest score
- Simple, beginner‑friendly logic using functions and dictionaries
