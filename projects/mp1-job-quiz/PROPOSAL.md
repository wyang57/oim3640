<!-- Mini Project 1 — PROPOSAL.md -->

<!-- What I'm building -->
I am building a Job Suitability Quiz: a Python program that asks a series of questions about a user’s preferences and strengths to eventually recommends a job category or role based on their answers.

<!-- Why I chose this -->
I’m currently exploring different career paths, and I would've appreciated a simple tool that helps clarify what types of roles align with my interests. This project is useful, easy to expand later, and a good way to practice input handling, scoring logic, with Python.

<!-- Core features -->
- Ask multiple-choice or scale-based questions about work preferences.
- Score the user’s answers across several categories (e.g., Data, Creative, Operations, Technical, Client-focused).
- Recommend a job type or role based on the highest score.
- Provide a short explanation of why that job type fits the user.
- Allow the user to retake the quiz without restarting the program.

<!-- What I don't know yet -->
- How I want to structure the scoring system (weights, categories, or point values).
- How to organize the code cleanly into functions.
- The amount of job/career paths I'll include
- Whether I want to add optional features like saving results to a file or offering different quiz modes.
