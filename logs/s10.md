"For vs While"
- "Use for": known sequence, fixed number of steps  
- "Use while": unknown number of steps, repeat until condition changes

- Print characters → "for"
- Roll until 6 → "while"
- Count vowels → "for"
- Ask until “done” → "while"

 uses_any(word, letters)
- Returns "True" on first match  
- Returns "False" if no match  
- Wrong placement of `return False` breaks the search

 version_a vs version_b
- "version_a": prints *all* vowels, then “Done”  
- "version_b": returns *first* vowel, stops immediately
